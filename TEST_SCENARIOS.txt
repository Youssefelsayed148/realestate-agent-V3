## NOTES FOR TESTING

1. Type each message exactly as written
2. Check that the bot extracts:
   - Unit type correctly
   - Location correctly
   - Budget (min/max when applicable)
   - Area (min/max when applicable)
   - Bedrooms
   - Features (garden, view, furnishing, etc.)
   - Intent (search, compare, filter, etc.)

3. The bot should respond appropriately based on the intent detected
4. Missing information should trigger follow-up questions
5. Complete searches should return matching results
